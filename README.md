# Install .NET SDK
- [Get started with .NET in 10 minutes](https://www.microsoft.com/net/learn/get-started/macos)

# Create new xunit project
```sh
dotnet new xunit -o dotnet-testing
cd dotnet-testing
```

# Run test
```sh
dotnet test
```
